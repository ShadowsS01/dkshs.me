A Django template with `cookiecutter` to quickly launch production-ready Django projects.

[![license mit](https://img.shields.io/badge/licence-MIT-56BEB8)](https://github.com/dkshs/cookiecutter-django/blob/master/LICENSE)
[![Code style: black](https://img.shields.io/badge/code%20style-black-000000.svg)](https://github.com/ambv/black)

## Usage

First, get Cookiecutter.

```bash
pip install cookiecutter
```

Now run it against this repo:

```bash
cookiecutter https://github.com/dkshs/cookiecutter-django
```

You'll be prompted for some values. Provide them, then a Django project will be created for you.

**Warning**: After this point, change 'DKSHS', etc to your own information.

Answer the prompts with your own desired [options](https://github.com/dkshs/cookiecutter-django/blob/master/docs/project-generation-options.md).
